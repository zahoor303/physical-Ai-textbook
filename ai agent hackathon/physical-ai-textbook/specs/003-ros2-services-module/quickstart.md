# Quickstart: ROS 2 Services Module

**Date**: 2025-11-29

This guide provides a quick overview of the "ROS 2 Services" module and how to get started with the content and examples.

## Prerequisites

-   You have a working ROS 2 Humble environment.
-   You have a basic understanding of ROS 2 nodes and topics.
-   You have cloned the `physical-ai-textbook` repository.

## Module Overview

This module introduces ROS 2 services, a fundamental request-response communication pattern. You will learn:
-   The conceptual difference between services and topics.
-   How to define your own custom service interface.
-   How to implement service servers and clients in both C++ and Python.
-   How to use command-line tools to interact with services.

## Getting Started

1.  **Navigate to the Module**: The content for this module is located in the `docs/module-2-services` directory. You can start by reading the `01-introduction-to-services.md` file.

2.  **Code Examples**: The code examples for this module are located in `static/code-examples/module-2-services`. You can use these examples to follow along with the tutorials in the module.

3.  **Running the Examples**:
    -   To run the examples, you will need to create a ROS 2 package and build it.
    -   Detailed instructions on how to create a package and build the code are provided within the module's content.
